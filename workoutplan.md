# 5/3/1 Forever Training Plan

## Overview

| Detail | Value |
| --- | --- |
| **Split** | Chest / Back / Shoulders / Legs (4-5 days/week) |
| **Main Lifts** | Follow 5/3/1 scheme (Chest, Back, Legs) |
| **Assistance Work** | Bodybuilding style, total ~18 sets + core |
| **Warmup** | Joe DeFranco's Limber 11 & Simple 6 |
| **Jumps/Throws** | 10-20 reps before or during warmup sets |
| **Training Max** | 85% of estimated 1RM |

* * *

## Protocols

### Leader: 5's PRO + BBB

- **Main Lift** - all sets kept at 5 reps (no AMRAP)
    - Week 1: 65% x5, 75% x5, 85% x5
    - Week 2: 70% x5, 80% x5, 90% x5
    - Week 3: 75% x5, 85% x5, 95% x5
- **Supplemental** - BBB 5x10 @ 60% (can use alternate lift)
- **Conditioning** - 1 day/week hard, other days easy

### Anchor: 5/3/1 PR Sets + 5x5 FSL

- **Main Lift** - original 5/3/1 AMRAP on last set
    - Week 1: 65% x5, 75% x5, 85% x5+
    - Week 2: 70% x3, 80% x3, 90% x3+
    - Week 3: 75% x5, 85% x3, 95% x1+
- **Supplemental** - 5x5 @ first set weight (FSL)
- **Conditioning** - 2 days/week hard, other days easy

### Conditioning

| Type | Details |
| --- | --- |
| Easy | 30-45 min walk, bike, etc. |
| Hard | Prowler - 10 x 40-yard sprints @ bodyweight, no time limit |
